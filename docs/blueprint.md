# **App Name**: FarmSight

## Core Features:

- Login Authentication: Secure login page for different user roles.
- Dashboard Overview: Display key metrics such as total items, low stock items, sales, and inventory value in an easy-to-understand dashboard.
- Inventory Management: Enable users to add, edit, and delete inventory items, categorized with details like quantity, unit, cost, and supplier.
- Sales Tracking: Record sales and usage transactions, tracking item name, quantity, transaction type, and total amount.
- Report Generation: Generate reports for inventory, sales, and profit/loss summaries within a specified date range.
- Notifications: Alerts for low stock items and expired products

## Style Guidelines:

- Primary color: Forest Green (#2E7D32) for buttons and highlights, symbolizing growth.
- Background color: Light Gray (#F8F9FA) to provide a clean and neutral backdrop.
- Accent color: Teal Green (#00BFA5) to highlight important elements and actions.
- Font: 'Inter', a sans-serif font, for body text and headers; ensures readability and a modern feel.
- Use 'React Icons' set (e.g., MdDashboard, FaBox, FiUsers, AiOutlineBell).
- Responsive layout with a fixed sidebar for navigation and collapsible elements for mobile devices.
- Subtle animations for loading states and form validations.
